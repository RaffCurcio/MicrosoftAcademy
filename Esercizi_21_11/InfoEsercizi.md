Il file Struct.cs contiene l'implementazione delle funzionalità richieste
nella lezione dalle 14:00 alle 18:00.
Vengono implementate funzionalità per interagire con una struct che contiene dati anagrafici.
In particolare, sono state aggiunte le seguenti funzionalità:
- Inserimento di nuovi dati anagrafici con controllo sugli input.
- Visualizzazione dei dati anagrafici esistenti.
- Aggiornamento dei dati anagrafici.
- Eliminazione dei dati anagrafici.
- Calcolo e visualizzazione di statistiche sui dati anagrafici

Il file Pro.cs contiene gli esercizi richiesti nella lezione dalle 9:00 alle 13:00.
In particolare sono esercizi basati sull'utilizzo di array. L'esercizio comprende:
- Stampa degli elementi di un array.
- Stampa degli elementi di un array in ordine inverso.
- Verifica dell'ordinamento di un array.
- Calcolo di statistiche in un array di elementi casuali.
- Conversione di numero intero ad ASCII.
